Tetris 2 Source

Description

This is the full source code for my game Tetris 2 for Dreamcast.
Included is also updated files for the sndmp3 lib that is included with KOS, the changes consist of bug fixes and the ability to add a callback function that will recive information about state changes, this is used to know when one track has ended and we need to start the next one. A XML parser is also included and this one is used for the theme system in the game.
Tetris 2 is built with KOS 1.2.0 and g++ 3.0.4 for SH4 processors.
