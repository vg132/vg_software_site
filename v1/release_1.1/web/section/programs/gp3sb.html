Gp3 Set Builder

Description

With Gp3 Set Builder you can quick and easy build track set files that you use with Rene 'SDI' Smit's GPxPatch - GpLap.
There is a bug in GPxPatch v1.0-1.3 that prevents you from having the track set file in any other directory then [GP3DIR]\Gpxpatch, this is fixed in GPxPatch v1.4.
