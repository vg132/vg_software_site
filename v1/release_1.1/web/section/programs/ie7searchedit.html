IE7 Search Engine Editor

Description

IE7 Search Engine Editor is a editor for the search engine dropdown box in IE7, with this program you can add, delete, move and change your search engine options.
This program is tested with IE7 build 7.0.5112 (Windows Vista Beta 1) and 7.0.5219 (Windows Vista build 5219), it may or may not work with newer builds of IE7 due to changes to the search engine format.
This program also require .NET version 2.0.
