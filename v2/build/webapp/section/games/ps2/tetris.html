Tetris

Description

My first game for PlayStation 2 is a port of my Dreamcast Tetris game. Its very simple and alot of functions have been removed and there is no sound, just plain and simple Tetris. This is more a tech demo then a game.
The game is tested with InLink 1.3.7 and a PAL PlayStation 2, but as far as I know it should also work from a CD or DVD.
