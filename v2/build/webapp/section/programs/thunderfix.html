Thunderfix

Description

Thunderfix will patch Thundertray and make it work with newer versions of Thunderbird.
Thundertray looks for a program named "thunderbird.exe" running on your computer and minimize this program to the systemtray but with the latest version of thunderbird the program is named "thunde~1.exe" and thundertray can't find this program. Thunderfix will fix this problem and make thundertray work with newer versions of thunderbird.
You have to run this program in the same folder as your thundertray.exe file.
