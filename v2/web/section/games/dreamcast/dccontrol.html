DCControl

Description

This is a wrapper class for KOS that helps get usable information about the state of the dreamcast controller.
The class is updated with the current state of the control on every main game loop loop and can be used to do somthing if the button has been held down for 8 loops or maybe for every 8'th loop or somthing like that, you have the power to decide :)
It can also be used to check if a button has just been pressed or just been released. It has support for all digital and analog buttons on the dreamcast control.
