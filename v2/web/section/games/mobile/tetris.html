Tetris

Description

Another port of my Dreamcast Tetris game, this time to devices that support J2ME but proberly only the Sony-Ericsson K750i as its coded very much for this phone.
Download the zip file, extract to the "other" folder on your phone and then in the file manager of your phone check the other folder and select install for the "j2me_tetris" application and have fun on your way to or from work.
