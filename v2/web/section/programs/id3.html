ID3 Tag Edit

Description

ID3 Tag Edit is a small and easy to use to that lets you edit the ID3 taggs in your MP3 files. The ID3 Tag is located at the bottom of a MP3 file and contains information about artist, album, track and some other information. This version has only support for ID3v1 and not the new ID3v2.
