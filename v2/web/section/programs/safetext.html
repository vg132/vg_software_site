SafeText

Description

SafeText is a notepad like text editor with added functions. Every document you open have its own tab, your can encrypt and decrypt documents with TripleDES encryption, convert the text to Base64 and calculate MD5 sums of the text.
This program was made as a assignment for a university course in Visual Basic.NET and you will need to have the .NET Framework installed on you computer to be able to run this program.
