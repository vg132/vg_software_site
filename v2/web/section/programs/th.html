Gp2 Track Handler

Description

Gp2 Track Handler is a track manager program for Gp2 tracks.
Features in the program

  • Change tracks and track information (name, country, laps)
  • Change menu track maps (bmp and gif support)
  • Edit track information
  • Backup tracks
  • Uninstall tracks
  • Edit points
  • Add your carset EXE files from Steven Young's GP2Edit to you track set and you will be able to export them at the same time as the tracks
  • Change track length in the menu's (+-1m) you will get the right average speed etc
  • Support of English, France, Dutch, Italian, US, German and Spanish versions of GP2 (all known version of GP2)
  • Export and Import best lap times for every track
  • Save best lap times in a database
  • And More...
